# sTask Documentation

sTask is the Evolution CMS background task manager for queued jobs, worker
configuration, task logs, progress tracking, uploads, downloads, and operational
maintenance tasks.

These localized filesystem docs are the preferred dDocs entrypoint. The older
Docusaurus pages under `docs/pages` and `docs/i18n` remain historical package
documentation, but dDocs and agents should start from the language folders below.

## Languages

- [English](en/README.md)
- [Ukrainian](uk/README.md)
- [Polish](pl/README.md)
- [German](de/README.md)
- [French](fr/README.md)

## Coverage

The guides cover the current EvoUI manager panel, task and log details, worker
tables, worker settings, built-in workers, task runner behavior, progress and
file contracts, asset and permission checks, and custom worker migration notes.
